## 📌 **Overview**

**MailStorm Pro** is a professional-grade email stress testing tool designed for **educational and authorized testing purposes only**. It allows you to send multiple emails with **randomized content**, **dynamic subjects**, and **multi-threaded performance**.

> ⚠️ **IMPORTANT**: This tool is for **YOUR OWN EMAIL TESTING** only. Unauthorized use is **ILLEGAL** and strictly prohibited.

## ✨ **Features**

| Feature | Description |
|:-------:|:------------|
| 📧 **Multi-Message** | 15+ pre-defined random message templates |
| 🎲 **Random Subjects** | Dynamic subjects with unique IDs |
| ⚡ **Fast Delivery** | Optimized SMTP connection |
| 📊 **Real-time Progress** | Live status updates for each email |
| 🎨 **Professional UI** | Colorful terminal interface |
| 🔄 **Auto-Retry** | Handles rate limiting gracefully |
| 📝 **Customizable** | Easy to add your own messages |
